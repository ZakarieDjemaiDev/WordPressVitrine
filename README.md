# Les Bistrots de Metz — Vitrine Projet WordPress

Projet réalisé dans le cadre d'un stage pour un client réel.  
Ce dépôt est une vitrine de réalisation : **le code source n'est pas publié** pour des raisons de confidentialité client.

## Contexte

Site vitrine multi-établissements conçu pour un groupe de restaurants messins, avec un objectif de visibilité locale, clarté de navigation et prise en main simple côté client.

## Ce que j'ai réalisé

- Maquettage UI sous Figma (desktop + mobile)
- Intégration WordPress (thème Blocksy)
- Personnalisation CSS
- Mise en place et configuration des plugins
- Optimisation SEO de base
- Structuration des contenus
- Documentation de prise en main pour le client

## Stack / Outils

- WordPress
- Blocksy
- CSS
- Figma
- Plugins (SEO, sécurité, cache, formulaires, avis, sauvegarde, optimisation images)
